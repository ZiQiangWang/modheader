## Usage

```shell
npm install modheader
```

or

```shell
yarn add modheader
```

Import `dist/modheader.min.js` to your project:

```html
<script src="path/to/modheader.min.js"></script>
<script>
  modheader.init();
</script>
```

or use cdn instead:

```html
<script src="//cdn.jsdelivr.net/npm/modheader/dist/modheader.min.js"></script>
<script>
  modheader.init();
</script>
```
